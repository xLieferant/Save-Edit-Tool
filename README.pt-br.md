# ETS2 Save Edit Tool

[Deutsch Version](README.de.md) | [English Version](README.en.md) | [Versão em Português](README.pt-BR.md) 🔴

Uma ferramenta moderna e poderosa de edição de jogos salvos (*savegames*) para o Euro Truck Simulator 2 (ETS2) e, futuramente, American Truck Simulator (ATS).  
Status atual: **0.1 Alfa (em desenvolvimento)**

---

## Visão Geral
A ferramenta lê os jogos salvos do ETS2, exibe os dados, permite a edição e grava as alterações com segurança.  
Recursos ainda não disponíveis aparecem como "Em breve".

As atualizações são entregues por meio de um atualizador automático integrado que utiliza os *Releases* do GitHub.

## 🚀 Novidades na v0.2.0
No momento, estamos trabalhando em uma grande reformulação de UI/UX para tornar a experiência mais rápida e moderna.

- **UI Modernizada:** Layout limpo e estruturado com uma nova navegação baseada em abas.
- **Suporte a Temas:** Escolha entre três temas selecionados: *Neon Night*, *Dark Pro* e *Pure Light*.
- **Desempenho:** Lógica central otimizada para uma resposta mais ágil e tempos de carregamento mais rápidos.
- **Qualidade de Vida:** Gerenciamento de janelas modais aprimorado, textos de ajuda contextual e indicadores de status.
- **Foco na Comunidade:** Verificador de atualizações integrado e links diretos para feedback no GitHub.

## Recursos na Versão 0.1 Alfa

### UI / Janelas Modais
- Modais dinâmicos  
- Botões "Salvar" e "Cancelar" funcionais  
- Modais desativados com indicação "Em breve"  
- Navegação básica (Caminhão, Reboque, Perfil, Configurações)

### Caminhão
- Reparar caminhão  
- Editar nível de combustível  
- Trocar de tanque de combustível  
- Editar quilometragem  
- Editar placa  

### Reboque
- Reparar reboque  
- Editar placa do reboque  
- Editar peso da carga

### Perfil
- Editar XP  
- Editar dinheiro  
- Editar habilidades e experiência  
- Visualizar e modificar estatísticas (onde estiver estável)

### Configurações
- Tema de cores básico  
- Comboio 128  
- Valor do tráfego  
- Estacionar reboques duplos  
- *Placeholder* de idioma  
- Modo de desenvolvedor

## Instalação & Atualizações
Entregue como instalador MSI.  
O atualizador automático verifica os *Releases* do GitHub em busca de novas versões.

Apenas lançamentos publicados ativam as atualizações.  
*Commits* regulares **não** geram notificações de atualização.
