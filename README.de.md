# ETS2 Save Edit Tool  
Ein moderner, leistungsfähiger Savegame-Editor für Euro Truck Simulator 2 (ETS2) und später auch American Truck Simulator (ATS).  
Aktueller Status: **0.1 Alpha (in Entwicklung)**

---

## Übersicht
Dieses Tool liest ETS2-Spielstände aus, zeigt deren Werte an, erlaubt Bearbeitungen und schreibt die Änderungen zurück.  
Nicht verfügbare Features erscheinen als „Coming soon“.

Updates werden über einen Auto-Updater bereitgestellt, der GitHub Releases nutzt.

---

## Features in Version 0.1 Alpha

### UI / Modale
- Dynamische Modalfenster  
- Save & Cancel Logik  
- Deaktivierte Modale für „Coming soon“  
- Grundnavigation (Truck, Trailer, Profil, Settings)

### Truck
- Truck reparieren  
- Kraftstoffstand ändern  
- Kraftstofftank ändern  
- Kilometerstand bearbeiten  
- Kennzeichen ändern

### Trailer
- Trailer reparieren  
- Kennzeichen bearbeiten  
- Jobgewicht ändern

### Profil
- XP bearbeiten  
- Geld ändern  
- Skills und Erfahrungspunkte anpassen  
- Profilstatistiken anzeigen und ändern (sofern möglich)

### Settings
- Grundlegendes Farbtheme  
- Convoy 128 Option  
- Traffic Value  
- Parking Doubles  
- Sprache (Platzhalter)  
- Dev Mode

## Installation & Updates
Das Tool wird per MSI-Installer ausgeliefert.  
Der Auto-Updater lädt neue Versionen automatisch von GitHub Releases herunter.

Nur Releases triggern Updates.  
Normale Commits lösen **keine Updates** aus.
